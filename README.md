# SeaTemperature
In the present repository you will find the data, R scripts, documentation and visualizations used for the completion of the assignment "Practical R work" on the first part of the subject "Statistical Software: R & SAS" (group B) from the MESIO UPC-UB programme. 

Authors are: Enrique Akira Takiguchi (enrique.akira.takiguchi@estudiantat.upc.edu) and Itziar Salazar (itziar.salazar@estudiantat.upc.edu). Therefore, if you use this code or analysis in your own research, please attribute it.

This work was conducted under the supervision of Professor Antonio Monleon Getino (UB) during the Autumn Semester of the 2025-2026 academic year.

The repository is organized as follows:
- **`SeaTemperature.pdf`**: Report presenting the study and findings following the instructions found in `PracR_GrB_2025english.pdf`
- **`SeaTemperature.rmd`**: R Markdown file containing the main and comments that generate the report
- **`/code`**: Includes auxiliary R functions to correctly run the main code
  - `createDF.R`: Downloads the necessary data and generates returns two dataframes
  - `Ex3.R`: Generates different plots
- **`/data`**: Contains data in different formats which are either necessary for the execution of the main code or product of the execution itself
  - `cmems_mod_med_phy-sal_my_4.2km_P1M-m_1766852522604.nc`: Dataset containing information about the Mediterranean salinity, for more information check the report
  - `sea_data.RData`: Data generated by `createDF.R`
